# 📐 Бот — Задание №18 ЕГЭ по математике

## Структура файлов

```
math18_bot/
├── bot.py          — точка входа
├── config.py       — все настройки (токены, цена)
├── database.py     — SQLite: пользователи, задачи, платежи
├── handlers.py     — вся логика бота
├── keyboards.py    — кнопки
├── payment.py      — ЮКасса (СБП включён автоматически)
├── requirements.txt
└── images/         — сюда сохраняются картинки задач
```

---

## Быстрый старт

### 1. Установи зависимости
```bash
pip install -r requirements.txt
```

### 2. Настрой config.py
Открой `config.py` и замени:
- `YOUR_BOT_TOKEN` → токен от @BotFather
- `123456789` → твой Telegram user_id (узнать: @userinfobot)
- `YOUR_SHOP_ID` и `YOUR_SECRET_KEY` → данные из личного кабинета ЮКассы
- `@your_username` → твой телеграм для связи

### 3. Настрой ЮКассу
1. Зарегистрируйся на yookassa.ru
2. Для тестов — используй тестовые ключи
3. СБП включается в настройках магазина ЮКассы отдельно (раздел "Способы оплаты")
4. В production замени return_url на реальную ссылку на бота

### 4. Запусти бота
```bash
python bot.py
```

---

## Как добавлять задачи

Напиши боту `/admin` — откроется панель.
Там можно:
- Добавить задачу (с картинкой или без)
- Посмотреть статистику (пользователи, выручка)

Картинки сохраняются в папку `images/` автоматически.

---

## База данных

SQLite файл `math18.db` создаётся автоматически при первом запуске.

Таблицы:
- `users` — все пользователи, флаг `has_access`
- `tasks` — задачи (условие + разбор + картинки)
- `payments` — все платежи и их статусы

Самый простой вариант для старта:

На Railway:
1. Загрузи код на GitHub
2. Подключи репо к Railway

4. Готово!
